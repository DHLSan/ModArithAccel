# Reduction for a Fixed Prime

Barret Reduction vhd files for hardware working
Create a Bitstream file and upload it to the card. After that, we print the result of the operation on the screen with the button down

  After adding the constraint files and vhdl files as source, we create our bit file, then we upload it to our zedboard. Button down enable button, button center reset button. In order to print the result in our vhdl file that we created for the fixed prime number on the OLED screen, the enable button, that is, the button down button must be pressed.
